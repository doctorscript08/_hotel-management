🛏 Solteiro
Perfeito para viajantes individuais, o Quarto Solteiro oferece conforto e praticidade com uma cama individual, casa de banho privativa, Wi-Fi e ar condicionado.

❤️ Casal
Ideal para casais ou duas pessoas, o Quarto Casal dispõe de uma cama de casal espaçosa, decoração acolhedora e todas as comodidades essenciais para uma estadia relaxante.

👨‍👩‍👧 Familiar
Projetado para famílias, este quarto é mais amplo, com camas adicionais e espaço suficiente para crianças. Inclui zona de estar, TV, mini-frigorífico e casa de banho completa.

👨‍👨‍👦 Triplo
Com capacidade para três adultos, o Quarto Triplo oferece uma configuração versátil (três camas individuais ou uma de casal + uma individual), ideal para pequenos grupos ou famílias.

🛌🛌 Gêmeos
O Quarto Gêmeos possui duas camas individuais separadas, perfeito para colegas de trabalho ou amigos que viajam juntos, mantendo conforto e privacidade.

🏨 Suite
A opção mais luxuosa, a Suite oferece um espaço exclusivo com quarto, sala de estar, decoração premium, banheira, minibar e serviço personalizado para uma experiência de alto nível.
